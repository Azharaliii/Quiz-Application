# 📝 Interactive Quiz Application (Java)

A desktop-based Quiz Application built using **Java Swing** and **AWT**. This project provides a smooth user interface to conduct quizzes, track scores, and manage rules for a seamless testing experience.

## ✨ Features
* **User Authentication:** Simple login screen (`Login.java`) to start the session.
* **Interactive Quiz UI:** A clean interface for questions and multiple-choice options (`Quiz.java`).
* **Timer/Rules System:** Integrated rules and guidelines before starting the quiz (`Rules.java`).
* **Automatic Scoring:** Instant score calculation and display at the end (`Score.java`).
* **Visual Assets:** Custom branding with logos and backgrounds.

## 🛠️ Tech Stack
* **Language:** Java (JDK 8 or higher)
* **GUI Library:** Java Swing & AWT
* **IDE:** IntelliJ IDEA (includes `.iml` configuration)

## 📂 Project Structure
* `Login.java` - The entry point for user login.
* `Quiz.java` - Main logic for handling questions and user input.
* `Rules.java` - Displays instructions to the player.
* `Score.java` - Handles final result calculation and display.
* `*.jpg/png` - Graphical assets for the UI.
